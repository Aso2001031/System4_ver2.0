```uml
@startuml
opt ユーザー未登録
ユーザー -> webサーバー : ユーザー登録
webサーバー -> DBサーバー : ユーザー登録
DBサーバー -> DBサーバー : 登録処理
DBサーバー -> webサーバー : 登録結果

alt 登録成功
 webサーバー -> ユーザー : ログインページを表示
else 登録失敗
 webサーバー -> ユーザー : 失敗メッセージを表示
 end
end

opt ログイン
ユーザー -> webサーバー : ログイン
webサーバー -> DBサーバー : ログイン照会
DBサーバー -> DBサーバー : ログイン処理
DBサーバー -> webサーバー : 認証結果

activate ユーザー
alt 認証成功
webサーバー -> ユーザー : メインぺージを表示
else 認証失敗
webサーバー -> ユーザー : 失敗メッセージを表示
end

opt ログイン中

opt 会員情報
ユーザー -> webサーバー : 検索(ユーザ)
webサーバー -> DBサーバー : 検索処理(ユーザー)
DBサーバー -> DBサーバー : 検索処理
DBサーバー -> webサーバー : 検索結果
webサーバー -> ユーザー : ユーザー情報を表示
end

opt 会員情報編集
ユーザー -> webサーバー : 編集(会員情報)
webサーバー -> DBサーバー : 編集処理(会員情報)
DBサーバー -> DBサーバー : 編集処理
DBサーバー -> webサーバー : 編集結果
alt 編集成功
webサーバー -> ユーザー : 編集結果を表示
else 編集失敗
webサーバー -> ユーザー : 編集失敗メッセージを表示
end
end

opt 記録投稿
ユーザー -> webサーバー : 投稿(記録)
webサーバー -> DBサーバー : 投稿処理(記録)
DBサーバー -> DBサーバー : 投稿処理
DBサーバー -> webサーバー : 投稿結果
alt 投稿成功
webサーバー -> ユーザー : メインページを表示
else 投稿失敗
webサーバー -> ユーザー : 投稿失敗メッセージを表示
end
end

opt グループ入室
ユーザー -> webサーバー : 入室
webサーバー -> DBサーバー : 入室処理
DBサーバー -> DBサーバー : 入室処理
DBサーバー -> webサーバー : 入室結果
alt 入室成功
webサーバー -> ユーザー : 投稿一覧ページを表示
else 入室失敗
webサーバー -> ユーザー : 入室失敗メッセージを表示
end
end

opt ログアウト
ユーザー -> webサーバー : ログアウト
webサーバー -> DBサーバー : ログアウト照会
DBサーバー -> DBサーバー : ログアウト処理
DBサーバー -> webサーバー : ログアウト結果
webサーバー -> ユーザー : ログアウト結果
end


end

end

@enduml
```
